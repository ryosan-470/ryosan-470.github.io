# はじめに

2025年2月、私はついに人生で最大の買い物であるマイホームを購入しました。
(正確には新築なのでまだ引渡しは受けておらず、決済したわけではないですがめでたく売買契約を締結しました。)

私はただのしがないソフトウェアエンジニアであり、不動産のプロではありません。そんな初心者の私が専門外の領域でもある不動産を購入するにあたってどのようなプロセスを踏んでいったかできる限り時系列でまとめていきたいと思います。少しでも誰かの役に立てば幸いです。

# 住まい探しをはじめたきっかけ

結婚してから1年ちょっとが経ち1人の暮らしから2人の暮らしにも慣れてきたと思うようになったことや、今住んでいる賃貸は定期借家契約で2年ごとの更新となり、あと半年ちょっとで更新が迫ってくる中そろそろ安住の地を賃貸からマイホームにしてもいいんじゃないか、と思うようになったことが大きいです。

## 私たち夫婦のプロフィール・情報

- 私。30代前半。フルタイム勤務。ほぼフルリモートでオフィスに行くとしても品川。
- 妻。30代前半。フルタイム勤務。週2在宅で残りは出社。オフィスは千代田区内。
- 子なし (今後ほしい。) 車なし。
- 現況は駅から徒歩10分程度の築20年ほどの賃貸マンション。2LDKで55平米ほど。
- 予算は最大でも私自身が住宅ローンを組むことができる金額まで。

## 情報収集

まずは情報収集からです。最初は SUUMO 等のアプリで物件を見たり相場を見ていましたがそれ以上の行動には移していませんでした。
特に検討を開始したのが2024年の初夏頃からでこのタイミングはちょうど首都圏を中心に不動産価格が強烈に上昇していた時期でした。
都心6区だけでなくその他都下や都外の近郊物件も軒並み上昇しており正直日和るレベルでの高値です。
私たち夫婦は別に中古でもいいかなーと思っていましたが近所に新築マンションができると聞いて Web サイトに張り付きつつそれ以上の行動はしていませんでした。
そんなあるとき、同僚と飲みに行った時にマンションを買った友人と話をしたことで、とあるマンションはどうかということで紹介してもらいせっかくなのでモデルルームに行ってみることとしました。

## モデルルーム見学 (2024年09月下旬ごろ)

そのマンションは都内のそこまで有名ではない駅から徒歩10分程度にある新築マンションでした。
本当に場所くらいしか調べておらず、とりあえず行ってみるべという感じでモデルルームに行きました。
ちょうどこのときが第1期第1次の申し込みが来週からです。まだ間に合うのでひとまず事前審査をしてみましょう！と営業には急かされましたが、私たち夫婦はまだ本気で買うつもりもなく様子見というのもあったので申込等はしませんでした。
とはいえ営業に予算に関して不安があるという話をしたところ、ファイナンシャルプランナーを紹介しますよ、となり次の打ち合わせで実際にファイナンシャルプランナーとお話しすることになりました。

## ファイナンシャルプランナーとの打ち合わせ

ファイナンシャルプランナーとの打ち合わせはとても有意義でした。
もちろんファイナンシャルプランナーとしては私がこの物件を購入することで手数料が入るのでしょうがそこまでこの物件を推してくるような人でもなく割と中立に話してくれたのが好印象でした。
ファイナンシャルプランナーは私たちの家計や今後の将来を交えてさまざまなシミュレーションを組んでくれるので非常に助かりました。例えば、

- ペアローンを組んで2人とも最大の住宅ローンを借りる場合の上限額。
- 子供が何人できるかを想定しての生活費や教育費を加味した住宅ローンの上限額。

## 方向性を決めた

この時点で予算を含め物件探しの方向性を夫婦で話し合って決めることができました。
私たちの希望条件は以下の通りです。

- 予算
    - 現時点での預貯金額や生活水準からどこまで組んでも安全か数値で判断できるようになった。
- 広さや間取り
    - 3LDK以上。できたら 4LDK。
    - できたら、70平米以上。
- 戸建てかマンションか -> マンションに
    - 断熱性
        - 妻が冷え性でマンションだと冷え症が発症しないことに感動しているのがいちばんの理由。
    - 資産性
        - 将来的に10年程度で住み替えも視野に入れているので流動性が高いマンションにしよう。
    - 立地の良さ
        - どうしても戸建てだと駅から離れてしまう。車を持っていない我々にとっては駅遠は選択しづらかった。
    - ゴミ出し
        - 24時間ゴミ出しに慣れてしまった。
- 立地
    - 妻の職場から通いやすい場所。
        - 妻の職場は千代田区内で、私の職場は品川なので、どちらもアクセスしやすい場所が理想。
    - 駅から徒歩10分以内。
    - 現況から近所の方がいい説
        - 妻があんまり知らない街に住みたくないという。
        - 確かに実際これはあとで中古探しをしている時も近所の方が探しやすかった

## 中古マンションの探しの旅に出る (2024年10月中旬)

そんなこんなで方向性も決まり、モデルルーム見学をさせてもらった営業さんにお礼を言いつつ我々はこのマンションは検討外や！と言い放ちさようならしました。(営業さんごめんなさい。)
あまり新築マンションが販売されていないのと、新築でも中古でもどちらでもいい、というスタンスだったのではじめてこのタイミングで近所の中古マンションの内見に行きました。
まずはじめに何軒か SUUMO で気になった物件をピックアップして内見の申し込みをしました。するとすぐに不動産業者から連絡が来ました。何社か連絡を取り、個人的に連絡が早く要望に応じてくれた業者をメインに内見等の仲介をお願いしました。

### 学び

- 仲介業者は複数社と連絡を取ると良い
    - 自分の要望や連絡頻度とあう業者を選ぶとスムーズだった。
    - 仲介手数料は物件価格の3%+6万円程度であり、仮に5000万円の物件を購入すれば、150万円程度の手数料を支払うことになるのでしっかり仕事をしてくれる業者を見つけるべき。

### 中古マンション 内見(1)

今住んでいるマンションから徒歩10分くらいにあるマンションで住宅街の中にマンションが建っているという立地でした。
空室内見だったので賃貸マンションの内見のように自由に見れるのはよかったです。ただ実際に見てみるとところどころ傷だらけだったのと相場より意外と安いのにも関わらず私が SUUMO でお気に入りに入れてから3ヶ月ほど売れないのを見てあんまり人気がないのかなと思ったのと、ここは妻も特に気に入らなかったのでパスしました。
この時点で業者から一旦事前審査を出しませんか、と提案されました。事前審査とは住宅ローンの本審査の前に行う簡単な審査で収入証明等を提出して銀行からいくらくらい借りられるか確認できます。事前審査を行なっておくことで限界もわかります。特に私の場合は転職してから3ヶ月程度だったので実際にどこまで借りられるのか確認するためにもこの物件を購入つもりはありませんでしたが、事前審査を行いました。

### 中古マンション 内見(2)

このマンションは大規模マンションで、400〜500戸くらいのマンションでした。初めての居住中内件だったので知らない人の家にお邪魔するのは緊張しました。
ただこのマンションの売主さんは非常に良い方たちで、いろいろな質問にも答えてくれ好印象でした。
さらに室内の工夫やインテリアについても真似したいところがいくつかあったので非常に参考になりました。
さらに大規模マンションの醍醐味といってはなんですが、共用施設が非常に充実していて、キッズルームやパーティールーム、ゲストルームにカフェ、ジムなども併設していたのは非常に良い点でした。
さらにこの物件価格は割と安いのが魅力的でした。
ただ、近所にスーパー等の商業施設が少なく、駅は地下鉄1本という少し頼りないアクセスだったのと川が近くにあったので水害のリスクがあるのが気になりました。
この物件は非常に良い物件でしたが、主に私が上記の点で微妙だなと思ったことと、まだまだ内見を続けていきたいという気持ちがあったので候補に入れつつ他を見ることにしました。

### 中古マンション 内見(3)

このマンションは [中古マンション内見(2)](#中古マンション-内見2) のマンションと同じくらいの規模感のマンションでした。実はこのマンション自体は内見できなかったのですが、周囲を散歩して夫婦共にこのマンションの空きがでたらすぐに内見したい、というくらい気に入っていました。それくらいマンション周囲の環境や雰囲気には満足していました。ただ、物件が出て仲介業者経由で内見を申し込みしようとしてもズルズルと内見希望日をずらされて行きなかなか内見できませんでした。この物件は非常に人気があるのかわかりませんでしたが結局一度も内見できないまま他の買主に購入されてしまったようです。

### 中古マンション 内見(4)

このマンションは私たちが住んでいるエリアの最寄駅の反対側にあるマンションでした。私たちが周囲に住んでいることもあり遠出した他の物件に比べ周囲の環境を調査せずともわかっているので非常に安心感がありました。
この物件を内見するまでに気づいたこととして、人気のある物件はすぐに誰かが買い入れてしまうということです。そこでわたしは SUUMO で見かけた瞬間に仲介業者に電話し最短で内見を調整してもらいました。
当日、私たちは内見1番手で内見でき、今までの物件の中で一番よく、室内も良好でした。さらにこの物件は眺望がよく雰囲気も良かったです。そこで帰り際にこの物件を購入するという意思を伝えました。すぐに買い付けを入れたのですが、帰宅後に仲介業者から連絡が来て他の買主も買い付けを入れているとのことでした。ここで仲介業者から買上の提案を受けました。買上というのは価格を上乗せして購入するということです。私たちはこの物件を非常に気に入ったので買上をしましたが結局のところ他の買主に購入されてしまいました。
仲介業者からは私たちの契約希望日が2週間後だったのが原因だったと言われてしまいました。たしかにこの内見の1週間後の土日はどうしても外せない用事があり契約希望日を2週間後にせざるを得ないのでした。とはいえこれ以外にもりゆうがありそうです。おそらくもう1件の買主は売主と仲介業者が一緒なので両手仲介ができ利益を総取りできるのです。これが敗因だったのではないかと思いました。

### 中古マンション内見のまとめと学び

このあたりで10軒程度内見をし実際に買い入れを入れた物件が1軒ということを考えるとかなり満足のいく購入をすることは苦戦することが想像できました。さらに毎週土日が内見で潰れてしまうので非常に疲れました。そこで一旦アクティブに内見することはやめて休憩することにしました。ここまでで学んだことは以下の通りです。

- 内見は非常に疲れるし時間もかかる。
    - 特にせっかくの休日が内見で潰れてしまうのは痛い。
    - 1日に2件程度が限界。
- 図面で見てみて良いと思っても言ってみると雰囲気が微妙とかある
    - 図面や写真だけでなく実際に行ってみると雰囲気があわないなどがかなりありました。
    - 例えば大きな家具がある、または部屋が片付いておらず物がいっぱいある物件だとものすごく狭く圧迫感を感じる。
    - 幹線道路沿いの物件はバルコニーが煤だらけでものすごく汚い。
- かならず夫婦でいくべき
    - 男性と女性で見る点が違うようです。私の場合かなり使いやすいかなどの良し悪しで見てしまいますが、その分部屋の汚れや雰囲気などは妻の方が気にしており両者が合致しないと満足する物件は見つけられません。

## 新築マンションの見学

この頃、近所に建つという噂のマンションのホームページが更新され、Web 事前案内の申し込みが開始されました。
正直新築マンションは高価で私たちの予算では届かないものではないかと思っていましたが中古マンションで気にいる物件が見つからない以上、新築マンションが販売されるなら見てみるのも手です。そんなわけで私だけで事前案内を聞くことにしました。
事前案内で聞いてみたところ意外と何とか買えそうな価格であることや他にもいろいろな魅力があり早速妻に相談したところ妻からもOKが出たので早速モデルルーム見学の予約をしました。
ちなみに私は事前案内の時に手付金について聞いてみました。通常マンション購入時の手付金は10%程度が相場だとうです。ただ手付金を10%支払うのはかなり大きな金額になりこの間何かあった時に不安が残ります。そこで5%程度にできないか聞いてみたところ、事情が事情なので5%でも大丈夫です。ただし10%支払える人がいたらそちらを優先します、というようなことを営業から言われつつもどうやら5%でも大丈夫なようでした。
その後、モデルルーム見学を行い、実際の物件の間取りや設備を確認できました。また予定価格を改めて案内していただき、私たちの予算でも購入できそうなことがわかりました。この価格帯で事前審査も通り購入できることはわかったのでこのまま進めていくことにしました。
この後、営業が何とか抽選にならないようお客さんをいい感じに調整します！と言っていましたが結局抽選になり (希望していた部屋の倍率は2倍) 色々とあわあわしましたが無事に当選し売買契約を交わしました。この時も手付金は5%ですんだので用意する金額は少なくて済みました。現在はオプションの選定をしつつ引き渡しまで気長に待つ毎日です。久々に大きなものを買ってワクワクしている自分がいます。

# まとめ

このような感じで私たちの住まい探しは進んでいきました。最初は中古マンションを探していましたが、最終的には新築マンションを購入することになりました。ただ中古マンションを最初に検討していたおかげで不動産についての様々な知識や用語の意味を理解してから新築マンションの場に臨むことができたので非常に良かったと思います。
